[What is The Wizard's Tower?]

  • A closed source non-commercial freeware to download Trading Card Games card sets
  • Support for a wide variety of Trading Card Games (work in progress, software updates will be released)
  • Only using open (free) TCG APIs
  • Available for Windows, macOS and Linux, powered by Microsoft .NET 10
  • A hobby project released under an imaginary company GoldenAxeSoft
  • The software is provided "AS IS", free of charge

[What is The Wizard's Tower NOT?]

  • A commercial software with technical support
  • A fully featured download manager with advanced download features (resume download, ...)
  • I have no time to implement user's feature requests...

[Installation]

For Microsoft Windows, Apple macOS and GNU/Linux: Microsoft .NET 10 Desktop Runtime does NOT need to be installed because it's included in the executable.

Microsoft Windows

  1. Download the correct ZIP file for your CPU: win-x64 or win-arm64
  2. Check the Microsoft Windows system requirements in the Downloads table!
  3. Extract the ZIP file
  4. Run the program TheWizardsTower.Wpf.exe

Apple macOS

  1. Download the correct ZIP file for your CPU: osx-x64 (Intel Mac) or osx-arm64 (Apple Silicon M1/M2/M3/M4/M5 Mac)
  2. Check the Apple macOS system requirements in the Downloads table!
  3. Open Finder
  4. Extract the ZIP file
  5. Copy TheWizardsTower.Mac.app to your Apps folder
  6. Control + Left mouse click on TheWizardsTower.Mac.app and choose "Show Package Contents"
  7. Open folder "Contents" and go to folder "MacOS"
  8. Control + Left mouse click on TheWizardsTower.Mac (inside folder MacOS) and choose "Open With" > "Other..."
  9. On the Choose an application dialog box:
    1. Check the "Always Open With" check box (near the bottom)
    2. Enable "All Apps"
    3. In the search field enter "Terminal" and select it
    4. Click on the Open button.
  10. You now can run The Wizard's Tower by clicking the app in your Apps folder
  11. Follow these instructions: Apple Support
  12. The App wil now run by clicking the app in your Apps folder

A great companion App to view the card images of The Wizards's Tower on your Mac is FlowVision. Extract the ZIP file and copy the App to your Apps folder.

GNU/Linux

  1. Download the correct ZIP file for your CPU: linux-x64 or linux-arm64
  2. Check the GNU/Linux system requirements in the Downloads table!
  3. Extract the ZIP file
  4. Make the program executable in a terminal: chmod +x TheWizardsTower.Gtk
  5. Run the program TheWizardsTower.Gtk

!!! IMPORTANT WARNING: PLEASE DO NOT RUN MULTIPLE INSTANCES OF THE WIZARD's TOWER AT THE SAME TIME !!!
If you do this, the card sets servers are overloaded with requests and your IP address will be banned by the server. This will result in a permanent inability to run the program because the server will not respond anymore to requests made by your IP address.
